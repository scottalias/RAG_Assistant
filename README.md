# RAG Assistant

A Retrieval-Augmented Generation (RAG) application built with Python that answers questions using custom documents and large language models.

## Features

- Document ingestion and indexing
- Semantic search using vector embeddings
- Context-aware responses with an LLM
- Command-line interface
- Modular Python architecture

## Technologies

- Python
- FAISS
- Ollama
- Sentence Transformers

## Future Improvements

- Web interface
- Conversation memory
- Support for PDFs, Word documents, and text files
- Docker deployment

## Purpose

This project demonstrates practical AI engineering skills, including vector databases, embeddings, retrieval pipelines, prompt engineering, and LLM integration.
